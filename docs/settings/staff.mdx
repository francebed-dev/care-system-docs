# スタッフの編集

この画面では、スタッフを編集することができます。

## 新規登録

スタッフを新規登録する場合は、画面右上の＋アイコンをタップします。<br></br>
新規登録画面に遷移するので、次の内容を登録します。

- 氏名を入力する
- ユーザーIDを入力する(半角英数字8文字以上)
- パスワードを入力する(半角英数字8文字以上)
- 確認のためもう一度同じパスワードを入力する
- 管理者権限の有無を設定する

<Info>
管理者権限のあるスタッフは、**フロア**、**部屋**、**ベッド**、**患者**、**スタッフ**、**ご利用中の端末**、**システム連携**の編集を行うことができます。
</Info>

上記の内容を登録したら、下部にある保存ボタンをタップします。

## 登録内容の変更

すでに登録されているスタッフの情報を変更したい場合は、変更したいスタッフをタップします。<br></br>
編集画面に遷移するので、変更内容を入力します。<br></br>
下部にある保存ボタンをタップします。


## 削除

登録されているスタッフを削除したい場合は、削除したいスタッフをタップします。<br></br>
編集画面に遷移するので、画面右上のゴミ箱のアイコンをタップします。<br></br>
削除を確認するダイアログが表示されるので、削除する場合は、「削除」のボタンをタップしてください。

<Error>削除してしまうと、復元することはできません。</Error>
