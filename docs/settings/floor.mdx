# フロア

この画面では、ご利用の施設のフロアを登録することができます。

## 新規登録

ご利用のフロアを新規登録する場合は、画面右上の＋アイコンをタップします。<br></br>
新規登録画面に遷移するので、登録するフロアの名前を入力し、下部にある保存ボタンをタップします。

<Info>フロアの名前は「1F」「2F」などスタッフが識別しやすい名前にすることを推奨します。</Info>
