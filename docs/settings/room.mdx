# 部屋

この画面では、ご利用の施設の部屋を登録することができます。

<Warning>同じ名前の部屋を複数登録することはできません。</Warning>

## 新規登録

ご利用の部屋を新規登録する場合は、画面右上の＋アイコンをタップします。<br></br>
新規登録画面に遷移するので、その部屋があるフロアを選択して、登録する部屋の名前を入力し、下部にある保存ボタンをタップします。

<Info>部屋の名前は「1-1」や「202」などスタッフが識別しやすい名前にすることを推奨します。</Info>

## 登録内容の変更

すでに登録されている部屋の情報を変更したい場合は、変更したい部屋をタップします。<br></br>
編集画面に遷移するので、変更内容を入力して、下部にある保存ボタンをタップします。

## 削除

登録されている部屋を削除したい場合は、削除したい部屋をタップします。<br></br>
編集画面に遷移するので、画面右上のゴミ箱のアイコンをタップします。<br></br>
削除を確認するダイアログが表示されるので、削除する場合は、「削除」のボタンをタップしてください。

<Error>削除してしまうと、復元することはできません。</Error>
