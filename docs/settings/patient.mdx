# 患者

この画面では、患者を登録することができます。

## 新規登録

患者を新規登録する場合は、画面右上の＋アイコンをタップします。<br></br>
新規登録画面に遷移するので、次の内容を登録します。

- 氏名を入力する
- 患者が利用しているベッドを選択する
- プッシュ通知を利用する場合は「プッシュ通知の使用」をONにする

上記の内容を登録したら、下部にある保存ボタンをタップします。

## 登録内容の変更

すでに登録されている患者の情報を変更したい場合は、変更したい患者をタップします。<br></br>
編集画面に遷移するので、変更内容を入力します。<br></br>
最下部に「ステータスをリセット」のチェックボックスがあるので、リセットする場合はチェックボックスを選択します。<br></br>
ステータスをリセットすると、その患者の体重測定結果やアラートログなどが削除されます。<br></br>
下部にある保存ボタンをタップします。

<Info>ステータスのリセットが必要な場合は、患者の入退院による患者の入れ替わりの際が想定されます。</Info>

## 削除

登録されているベッドを削除したい場合は、削除したいベッドをタップします。<br></br>
編集画面に遷移するので、画面右上のゴミ箱のアイコンをタップします。<br></br>
削除を確認するダイアログが表示されるので、削除する場合は、「削除」のボタンをタップしてください。

<Error>削除してしまうと、復元することはできません。また、その患者の体重測定結果やアラートログなども削除されます。</Error>
