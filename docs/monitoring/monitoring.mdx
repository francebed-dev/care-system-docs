# モニタリング

この画面では患者の現在の状態をリアルタイムで表示することができます。<br></br>

<Image src="/assets/monitoring/monitoring.png" alt="Monitoring" height="500" />

表示されているものは以下の通りです。

1. 患者の現在の状態を示すアイコン
2. 患者が利用しているM2の設定
3. 患者の名前と部屋
4. 患者の体重(現在の体重・昨日の体重)
5. 患者のバイタル情報(心拍数・呼吸数)
6. 背上げ・足上げ・高さの情報

<Info>患者のバイタルの表示には、別途オプションセンサーが必要です</Info>

## 患者の詳細を表示する

患者の詳細情報を表示したい場合は、その患者をタップします。<br></br>
画面下部からボトムシートが表示されます。<br></br>
ボトムシートには、患者が利用しているベッド等を操作するためのボタンや、患者の情報を簡易的に表示されています。

<Image src="/assets/monitoring/monitoring-bottom-sheet.png" alt="Monitoring-Bottom-Sheet" height="500" />

初期状態では、ボトムシートは画面の半分程度まで表示されます。<br></br>
フルスクリーンで表示するためには、ボトムシートを上方向にスワイプすると、ボトムシートがフルスクリーンで表示されます。<br></br>
ボトムシートを閉じる場合は、ボトムシートを下方向にスワイプするか、左上に表示されている×マークのアイコンをタップすることでも、閉じることができます。

<Image src="/assets/monitoring/monitoring-bottom-sheet-full.png" alt="Monitoring-Bottom-Sheet-Full" height="500" />

患者の詳細情報を表示するためには、「詳細」ボタンをタップします。<br></br>

<Image src="/assets/monitoring/monitoring-patient-detail.png" alt="Monitoring-Patient-Detail" height="500" />

画面上部の「アラートログ」タブをタップすることで、その患者のアラートログを表示することもできます。

<Image src="/assets/monitoring/monitoring-patient-alert-log.png" alt="Monitoring-Patient-Alert-Log" height="500" />

## アラートON

患者のアラートがONになると、患者のアイコンが点滅表示されます。<br></br>
また、右下にベルのアイコンが表示され、アラートがONになっている患者のリストを表示することができます。<br></br>
患者の状態を確認し、アラートをOFFにしたい場合は、確認OKボタンをタップすることで、アラートをOFFにすることができます。
